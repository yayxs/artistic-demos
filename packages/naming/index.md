### BEM

模块 b
元素 e
修饰 m

```sh
.user-home-nav 模块
.user-home-nav__icon 模块的子元素
.user-home-nav__text
user-home-nav__text--small 模块的变体
```

样式名不能包含 ad、guanggao、ads、gg 是广告含义的关键词，避免元素被网页拓展、插件屏蔽

```
card__title-big 卡片__元素-大
card__title-small
```

使用长的命名 .some-desc
